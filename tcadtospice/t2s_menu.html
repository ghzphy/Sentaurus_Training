TCAD to SPICE

The TCAD to SPICE products are designed to enable SPICE-level circuit analysis based on TCAD data, which is a critical component of the design-technology cooptimization (DTCO) flow. The tool flow includes methods for including process (global) and statistical (local) variability in SPICE models, as well as simulation-based process corners.

This module is a basic introduction to using the TCAD to SPICE products.
