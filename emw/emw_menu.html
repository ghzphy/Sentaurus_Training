Sentaurus Device Electromagnetic Wave Solver

Sentaurus Device Electromagnetic Wave Solver is a simulation module for the numeric analysis of electromagnetic waves. It is a full-wave time-domain simulator based on the finite-difference time-domain method. It is used typically to calculate the optical carrier generation in optoelectronic devices.

This module is a basic introduction to the features of Sentaurus Device Electromagnetic Wave Solver.
