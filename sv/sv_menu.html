Sentaurus Visual

Sentaurus Visual is a plotting tool for visualizing data from TCAD simulations and experiments. Users can work interactively with data using a graphical user interface and can automate tasks by using Tcl-based or Python-based scripting. Sentaurus Visual is part of Sentaurus Workbench Visualization.

This module is a basic introduction to the features of Sentaurus Visual.
