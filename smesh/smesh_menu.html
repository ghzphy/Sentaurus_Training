Sentaurus Mesh

Sentaurus Mesh is a modular Delaunay 2D and 3D mesh generator that can create axis-aligned, boundary-conforming, and tensor meshes to be used in simulators that use the box discretization or finite-difference time-domain methods.

The axis-aligned mesh generator of Sentaurus Mesh produces triangles in the case of 2D devices and tetrahedra in the case of 3D devices.

The tensor-product mesh generator produces rectangular elements in two dimensions and cuboidal elements in three dimensions.

This module is a basic introduction to the features of Sentaurus Mesh.
